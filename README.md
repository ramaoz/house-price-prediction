# House Price Prediction

This project predicts house prices using machine learning.

Model used:
Linear Regression (Scikit-Learn)

Features:
- OverallQual
- GrLivArea
- GarageCars
- TotalBsmtSF
- FullBath
- YearBuilt

Evaluation:
RMSE ≈ 39,700  
R2 Score ≈ 0.79

The model predicts house prices and visualizes predicted vs actual values.
